# Certificate Upload Guide

## 📁 Upload Location
Upload all certificate images to: `assets/certificates/`

## 📋 Required Certificate Files

### Internship Certificates:
1. `codetechnology-cert.jpg` - CodeTechnology internship certificate
2. `codealpha-cert.jpg` - CodeAlpha internship certificate  
3. `internpe-cert.jpg` - InternPe internship certificate
4. `matpo-cert.jpg` - MaTPO internship certificate
5. `greenskills-cert.jpg` - Green Skills & AI internship certificate

### Course Certificates:
1. `nptel-cert.jpg` - NPTEL course certificate
2. `aicte-internship-cert.jpg` - AICTE internship certificate

## 📝 File Format Requirements:
- **Format**: JPG, PNG, or PDF
- **Size**: Recommended max 2MB per file
- **Resolution**: At least 800x600 pixels for good quality
