### Colors and Themes
Edit CSS variables in `css/styles.css`:
```css
:root {
    --primary-color: #1a237e;
    --secondary-color: #2962ff;
    --accent-color: #00bcd4;

```

### Content Updates
- Update personal information in all HTML files
- Replace placeholder images with actual photos
- Modify project details and links
- Update social media links
- Customize contact information

### Adding New Projects
1. Add project card HTML in `projects.html`
2. Include project thumbnail in `assets/images/project-thumbs/`
3. Set appropriate `data-category` for filtering
4. Update project statistics if needed
